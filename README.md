## 实践思路
- 浏览器插件基于html,css,js技术
- 主要原理是在一个文件夹里面建立`manifest.json`文件，然后进行各项配置
- 然后在浏览器里加载这个插件
- 在url或者dom满足设置的约束条件时就会加载相应的脚本
